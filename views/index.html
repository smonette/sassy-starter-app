Sass Starter Kit

What is Sass?

Sass is a CSS precompiler. It gives you some functionality that isn't available in CSS out of the box.Things like nested code blocks, iteration, mixins. Sass also give you the ability to use partials to break apart your CSS file into smaller managable chunks, which is a game changer for most any codebase.

The best part is that Sass is just CSS. No need to learn a new language.

How to use this starter

This starter app is Hello Express that's been supercharged with Sass.

  1. Remix this app.
  2. Open source/style.scss.
  3. Try changing the colors to see what happens!
  4. Change colors, fonts, and other styles to match your app!
  5. Create new partials to keep your CSS managable
